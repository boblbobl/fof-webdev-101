HTML (repetition)

Hvad er HTML?

  • HyperText Markup Language
  • Et dokument der ofte ender med .html eller .htm
  • Standard sprong til at beskrive hjemmesider
  • HTML definerer sidens struktur og semantik via HTML elementer
  • HTML elementer fortæller browseren hvordan indhold skal vises

Eksempel på HTML Dokument


                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Sidens Titel</title>
                        </head>
                        <body>
                            <h1>Overskrift</h1>
                            <p>Dette er et afsnit.</p>
                        </body>
                        </html>

Hvad er et HTML Element?

Et HTML element består ofte af et start "tag", noget indhold, og et slut "tag"


                        <body>
                            <h1>Overskrift</h1>
                            <p>Dette er et afsnit.</p>
                        </body>

Her er <h1> start tag og </h1> slut tag.

Eksempler på HTML Elementer


                        <h1>Overskrift 1</h1>
                        <h2>Overskrift 2</h2>
                        <h3>Overskrift 3</h3>
                        <h4>Overskrift 4</h4>
                        <h5>Overskrift 5</h5>
                        <p>
                            Her er et afsnit med et link.
                            <a href="https://www.fof.dk/da/aarhus">
                                Link til FOF Aarhus
                            </a>
                        </p>

Andre Semantiske HTML Elementer


                        <html>
                        <body>
                            <header>
                                <nav></nav>
                            </header>
                            <main>
                                <section></section>
                                <article></article>
                                <aside></aside>
                            </main>
                            <footer></footer>
                        </body>
                        </html>

Ikke-Sematiske HTML Elementer


                        <div id="1234">
                            Her er et område af hjemmesiden isoleret via 
                            <span class="highlight">ikke-semantiske tags</span>
                        </div>

Ikke-semantiske tags bruges ofte af udviklerer til at "styre" indholdet af en hjemmeside, hvor semantiske tags ikke er tilstrækkelige

"Tomme" HTML Elementer


                        <img src="mig.jpg" alt="Billede af mig">
                        <meta charset="UTF-8">
                        <br>
                        <hr>

CSS

Hvad er CSS

  • Cascading Style Sheet
  • Bruges til at beskrive præsentation af et HTML dokument
  • Inkluderes CSS can inkluderes in-line, internt via <style> element i header eller via eksterne stylesheet referencer
  • Styling sker via CSS Syntax Selectors og Properties
  • Styling Kaskaderer/Nedarves - styling kan overskrives via præcedens

CSS IS AWESOME

Browseren anmoder webserveren via HTTP

Primære elementer der udgør en hjemmeside

Simlpe CSS Selectors


                        div {
                            color: blue;
                        }

                        #1234 {
                            color: blue;
                        }

                        .highlight {
                            color: blue;
                        }

Eksempel på HTML element, id og class selectors.

Simple CSS Selectors (fortsat)


                        h1, h2, h3, p {
                            color: red;
                        }

                        * {
                            color: red;
                        }

Eksempler på gruppering og universal (wildcard) selectors.

Combinators CSS Selectors


                        div p {
                            color: yellow;
                        }

                        div > p {
                            color: yellow;
                        }

Eksempler på descendant og child selectors.

Combinators CSS Selectors (fortsat)


                        div + p {
                            color: green;
                        }

                        div ~ p {
                            color: green;
                        }

Eksempler på adjacent - og general sibling selectors.

Pseudo-Class CSS Selectors


                        /* ikke besøgt link */
                        a:link {
                            color: blue;
                        }
                        /* besøgt link */
                        a:visited {
                            color: red;
                        }
                        /* mouse over link */
                        a:hover {
                            color: yellow;
                        }
                        /* klikket link */
                        a:active {
                            color: green;
                        }

Andre pseudo-class eksempler omfatter bl.a. :first-child og :lang

CSS Properties


                        div {
                            color: #ff0000;
                            padding: 10px auto;
                            margin: 10px 20px 10px 20px;
                            border: 2px solid #00ff00;
                            border-radius: 5px;
                            height: 200px;
                            width: 200px;
                        }

Flere CSS Selectors og Properties

  • Pseudo-element selectors (::first-line, ::first-letter, ::before, ::after)
  • Attribute CSS Selectors (a[target="_blank"])
  • Absolute længder - cm, mm, in, px, pt, pc
  • Relative længder - em, rem, %
  • !important
  • Matematik funktioner - calc(), max(), min()
