Tooling (repetition)

Hvad er Visual Studio Code?

  • Visual Studio Code (VS Code) er en IDE
  • Modsat en alm. tekst editor (notepad eller lign.) kan VS Code understøtte en række sprog
  • Lynhurtig og nemt at udvide via Extensions
  • Ud-af-boksen uderstøttes HTML, CSS, JavaScript m.fl.

Hvad er NPM?

  • Node Pakage Management
  • Standard pakke manager til Node.js (server-side scripting)
  • Command-line klient (npm)
  • Online database af pakker
  • Vi vil primære bruge den til at beskrive vores løsning
  • ... og køre lokalt en udvikligswebserver

Hvad er Git

  • Versionsstyring
  • Linus Torvalds i 2005
  • Distribueret
  • Sammen med GitHub vil vi bruge Git til gemme/dele kode samt hosting af vores hjemmeside

Git+GitHub

Hvad er Git (fortsat)?

  • Distribueret Versionsstyring
  • Git tager et "billede" af hvordan dit projekt ser ud på et givent tidspunkt
  • Git gemmer kun ændringer
  • Git er hurtigt
  • arbejder med branches, commits, merges

Git Versioner

Hvad er GitHub?

  • Live i 2008
  • Tilbyder et centralt sted på Internettet til din kode/kodehistorik
  • Gør det lettere at arbejde med kode i fælleskab
  • Projektstyring/Automation/Sikkerhed
  • Købt af Microsoft i 2018
  • Ikke den eneste (BitBucket, GitLab), men klart den mest populære
  • Bliver brugt af ~100 millioner udviklere

GitHub Funktioner

  • Pull Requests
  • Issues
  • Actions
  • Pages

GitHub er Gratis

  • Ubegrænset private/public kode biblioteker
  • 2.000 CI/CD min/mdr
  • 500 MB lagerplads til pakker
  • Ny issues og projects
  • Community Support
  • Automatiske sikkerheds- og version updates
