Hvad er en hjemmeside?

Det hele starter her

En bruger indtaster en adresse i browserens adressefelt, og hvad sker der så?

Browseren anmoder webserveren via HTTP

Web Server

  • Som udgangspunkt leverer Web Serveren HTML retur til Browseren
  • HTML beskriver hvilke andre ressourcer den skal anmode (CSS, JS, JPG, ...)
  • Web Server kan være dynamiske (PHP, ASP, ...)
  • HTTP (HyperText Transfer Protocol)

Primære elementer der udgør en hjemmeside

HTML

Hvad er HTML?

  • HyperText Markup Language
  • Et dokument der ofte ender med .html eller .htm
  • Standard sprong til at beskrive hjemmesider
  • HTML definerer sidens struktur og semantik via HTML elementer
  • HTML elementer fortæller browseren hvordan indhold skal vises

HTML på en tidslinje...

Eksempel på HTML Dokument


                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Sidens Titel</title>
                        </head>
                        <body>
                            <h1>Overskrift</h1>
                            <p>Dette er et afsnit.</p>
                        </body>
                        </html>

Hvad er et HTML Element?

Et HTML element består ofte af et start "tag", noget indhold, og et slut "tag"


                        <body>
                            <h1>Overskrift</h1>
                            <p>Dette er et afsnit.</p>
                        </body>

Her er <h1> start tag og </h1> slut tag.

Kan I se andre HTML elementer i ovenstående kodeeksempel?

Eksempler på HTML Elementer


                        <h1>Overskrift 1</h1>
                        <h2>Overskrift 2</h2>
                        <h3>Overskrift 3</h3>
                        <h4>Overskrift 4</h4>
                        <h5>Overskrift 5</h5>
                        <p>
                            Her er et afsnit med et link.
                            <a href="https://www.fof.dk/da/aarhus">
                                Link til FOF Aarhus
                            </a>
                        </p>

Andre Semantiske HTML Elementer


                        <html>
                        <body>
                            <header>
                                <nav></nav>
                            </header>
                            <main>
                                <section></section>
                                <article></article>
                                <aside></aside>
                            </main>
                            <footer></footer>
                        </body>
                        </html>

Ikke-Sematiske HTML Elementer


                        <div id="1234">
                            Her er et område af hjemmesiden isoleret via 
                            <span class="highlight">ikke-semantiske tags</span>
                        </div>

Ikke-semantiske tags bruges ofte af udviklerer til at "styre" indholdet af en hjemmeside, hvor semantiske tags ikke er tilstrækkelige

"Tomme" HTML Elementer


                        <img src="mig.jpg" alt="Billede af mig">
                        <meta charset="UTF-8">
                        <br>
                        <hr>

Visual Studio Code

Hvad er Visual Studio Code?

  • Visual Studio Code (VS Code) er en IDE
  • Modsat en alm. tekst editor (notepad eller lign.) kan VS Code understøtte en række sprog
  • Lynhurtig og nemt at udvide via Extensions
  • Ud-af-boksen uderstøttes HTML, CSS, JavaScript m.fl.
