JavaScriptm (repetition)

Primære elementer der udgør en hjemmeside

Browseren anmoder webserveren via HTTP

Hvad er JavaScript?

  • High-level intrepreted (fortolket) programmeringssprog
  • Kan bruges til at tilføje interaktivitet til din hjemmeside
  • JavaScript ≠ Java
  • Libraries og Frameworks (JQuery, Bootstrap, D3, GSAP, Threes.js, React, ...)
  • StackOverflow’s 2022 Udvikler Spørgeskema viser at 67.9% af alle professionelle udvikler arbejder med JavaScript

Hvad kan man med JavaScript?

  • Kan manipulere HTML elementer
  • Kan interagere med browseren
  • AJAX kald (Asyncronious JavaScript And XML)
  • Med hjælp fra indbygget metoder som Date/Math/Strings/Arrays m.fl.

AJAX kald via JavaScript

Hvordan bruges JavaScript så på en hjemmeside?

  • Indsættes direkte på elementet
  • Indsættes via <script> element i header
  • Indsættes via ekstern JavaScript fil

Metoder på Arrays (2)

                            // Tilføjer et element til slutningen af arrayet
                            fruits.push("Kiwi");

                            // Fjerner et element fra slutningen af arrayet
                            fruits.pop();

                            // Tilføjer et element til starten af arrayet

Local Storage

                            localStorage.setItem(key, value);

                            // Henter en værdi fra local storage

                            // Sletter en værdi fra local storage
                            localStorage.removeItem(key);

                            // Sletter alt fra local storage
                            localStorage.clear();
