HTML+CSS+Bootstrap (repetition)

Primære elementer der udgør en hjemmeside

Browseren anmoder webserveren via HTTP

Hvad er HTML?

  • HyperText Markup Language
  • Et dokument der ofte ender med .html eller .htm
  • Standard sprong til at beskrive hjemmesider
  • HTML definerer sidens struktur og semantik via HTML elementer
  • HTML elementer fortæller browseren hvordan indhold skal vises

Eksempel på HTML Dokument


                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Sidens Titel</title>
                        </head>
                        <body>
                            <h1>Overskrift</h1>
                            <p>Dette er et afsnit.</p>
                        </body>
                        </html>

Hvad er et HTML Element?

Et HTML element består ofte af et start "tag", noget indhold, og et slut "tag"


                        <body>
                            <h1>Overskrift</h1>
                            <p>Dette er et afsnit.</p>
                        </body>

Her er <h1> start tag og </h1> slut tag.

Eksempler på HTML Elementer


                        <h1>Overskrift 1</h1>
                        <h2>Overskrift 2</h2>
                        <h3>Overskrift 3</h3>
                        <h4>Overskrift 4</h4>
                        <h5>Overskrift 5</h5>
                        <p>
                            Her er et afsnit med et link.
                            <a href="https://www.fof.dk/da/aarhus">
                                Link til FOF Aarhus
                            </a>
                        </p>

Andre Semantiske HTML Elementer


                        <html>
                        <body>
                            <header>
                                <nav></nav>
                            </header>
                            <main>
                                <section></section>
                                <article></article>
                                <aside></aside>

Ikke-Sematiske HTML Elementer


                        <div id="id-1234">
                            Her er et område af hjemmesiden isoleret via 
                            <span class="highlight">ikke-semantiske tags</span>
                        </div>

Ikke-semantiske tags bruges ofte af udviklerer til at "styre" indholdet af en hjemmeside, hvor semantiske tags ikke er tilstrækkelige

"Tomme" HTML Elementer

Hvad er CSS

  • Cascading Style Sheet
  • Bruges til at beskrive præsentation af et HTML dokument
  • Inkluderes CSS can inkluderes in-line, internt via <style> element i header eller via eksterne stylesheet referencer
  • Styling sker via CSS Syntax Selectors og Properties
  • Styling Kaskaderer/Nedarves - styling kan overskrives via præcedens

Simlpe CSS Selectors


                        div {
                            color: blue;
                        }

                        #id-1234 {
                            color: blue;
                        }

                        .highlight {
                            color: blue;
                        }

                        h1, h2, h3, p {
                            color: red;
                        }

Combinators CSS Selectors


                        div p {
                            color: yellow;
                        }

                        div > p {
                            color: yellow;
                        }

Eksempler på descendant og child selectors.

CSS Properties


                        div {

                            margin: 10px 20px 10px 20px;
                            border: 2px solid #00ff00;
                            border-radius: 5px;
                            height: 200px;
                            width: 200px;
                        }

Hvad er Bootstrap?

  • Bootstrap 5
  • Open Source CSS Framework
  • JavaScript til interaktivitet
  • Omfattende Dokumentation
  • "bootstrape" hjemmesider med standarder man bruger fra gang til gang
  • den mest populære, men langt fra den eneste (f.eks. Bulma, Foundation, Pure.css)

Boostrap - Grundstenene

  1. Layout
  2. Komponenter
  3. Interaktivitet
  4. Hjælpeværktøjer (til styling)
  5. Temaer

Fordele ved Bootstrap

  1. Konsistent
  2. Letvægts (>35kb)
  3. Responsive hjemmeside
  4. Stort fællesskab
  5. Skabeloner
  6. Browser support
